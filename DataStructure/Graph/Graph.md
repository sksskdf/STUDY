그래프는 정점과 그 사이를 잇는 간선으로 이루어져 있는 데이터 구조이다. 
일반적으로 정점은 객체를 표현하고 간선은 객체간의 관계를 나타낸다.

주로 Adjacency Matrix(인접 행렬)와 Adjacency List(인접 리스트) 로 그래프를 표현한다.

인접 행렬은 그래프의 노드를 2차원 배열 형태로 표현한다.
행렬의 각 요소 m[i][j]는 각 노드가 서로 연결되어 있으면 1, 아니면 0으로 표현된다.
만약 그래프가 가중치를 가지느 경우 0이나 1이 아닌 가중치로 표현된다.
탐색 시엔 시간복잡도가 O(1)이지만 그래프 크기가 클 경우 연결되어 있지 않은 노드 쌍에 대해서도 저장하기 때문에 공간 효율성이 떨어진다.

인접 리스트는 인접한 노드들의 리스트를 저장하는 방법이다.
그래프의 크기가 큰 경우에도 공간 효율적이다.
하지만 탐색 시 해당 노드의 인접 리스트를 탐색해야 하므로 인접 행렬보다는 시간복잡도가 높다.

Directed Graph(유향 그래프)는 간선에 방향이 있는 그래프이다. digraph라고도 불리며 유향 간선 그래프에선 각선에 방향이 있기 때문에 head와 tail 개념이 사용된다.
tail은 간선이 출발하는 노드, head는 도착하는 노드이다.

Undirected Graph는 간선에 방향이 없는 그래프이다.

Spanning Tree는 무향 간선 그래프를 기반으로 모든 노드를 연결하며 간선의 수는 최소한으로 사용하여 사이클이 없는 구조를 말한다.
하나의 무향 간선 그래프에는 여러개의 신장트리가 있을 수 있다.

Minimum Spanning Tree는 무향 간선 그래프를 기반으로 하나, 간선에 가중치가 존재 할 경우 간선의 가중치 합이 최소인 신장트리를 말한다.
